## Micro-copy Principles

### âœ… DO:
1. **Use second person ("you/your")** - Creates intimacy
2. **Quantify everything** - "Save 20 hours" not "Save time"
3. **Use active voice** - "You saved R5,000" not "R5,000 was saved"
4. **Show progress** - "2 of 5 complete" not "3 remaining"
5. **Acknowledge emotions** - "We know rostering is stressful"
6. **Use SA terminology** - "Guards" not "Security Officers"
7. **Include time estimates** - "This takes 2 minutes"

### âŒ DON'T:
1. **Don't use jargon** - "PSIRA reg. number" not "Regulatory compliance identifier"
2. **Don't blame users** - "Let's try again" not "You entered the wrong format"
3. **Don't hide costs** - Always show "(excl. VAT)"
4. **Don't overwhelm** - Show 3 options max, not 10
5. **Don't ignore context** - New users see different messages than power users

## Error Handling Philosophy

**Principle:** Errors are learning opportunities, not failures.

### Error Message Template:
```
[Friendly acknowledgment] + [What happened] + [Why it matters] + [How to fix]

Example:
"Oops! That ID number is only 12 digits.

South African IDs are always 13 digits, which helps us verify PSIRA compliance.

Can you double-check and try again?"
```

### Error Severity Levels:

**ðŸŸ¢ Informational (Green)**
- "Just so you know: This guard hasn't logged in yet"
- Action: None required
- Tone: Helpful, friendly

**ðŸŸ¡ Warning (Yellow)**
- "Heads up: This roster has 2 uncovered shifts"
- Action: Suggested but optional
- Tone: Concerned, supportive

**ðŸ”´ Critical (Red - use sparingly)**

## Cultural Considerations (South Africa)

### Language:
- **Afrikaans speakers:** Tend to prefer direct, no-nonsense copy
- **English speakers:** More receptive to friendly, casual tone
- **Solution:** A/B test tone based on language selection

### Trust Building:
- Emphasize local data storage
- Show South African team photos
- Use ZAR currency exclusively
- Reference local regulations (BCEA, PSIRA)

### Social Proof:
- Use real South African company names (with permission)
- Show city locations (Johannesburg, Cape Town, Pretoria)
- Use realistic guard counts (20-200 range)
